# Jarvis

Jarvis is your conversational assistant available as a console programm. It can be used all local. The software consists of client and server.

- Voice To Text 📝:

  - faster-whisper(local)
  - whisper
  - google gtts

- Brain 🧠:

  - ollama(local)
  - groq
  - chatGPT

- Text to Speech 💬:

  - piper (local)
  - xtts (local)
  - google gtts
